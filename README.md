# Semester Grade Calculator

A comprehensive semester grade calculator developed as an academic project. This repository contains a practical implementation of a modern web application for managing weighted grades, tracking GPA, and monitoring semester progress in a clean and responsive interface.

## 📁 Project Structure

### 1. Web - Semester Grade Calculator
Located in the root project folder, this is an interactive semester grade calculator built with Next.js and Tailwind CSS.

**Features:**

- Weighted grade calculation for each course
- Live GPA update based on course credits
- Visual semester progress bar
- Add and remove courses dynamically
- Clean and mobile-responsive user interface

**Tech Stack:**

- Next.js
- React
- Tailwind CSS
- TypeScript

## 🚀 Getting Started

### Prerequisites

- Node.js 18 or higher
- npm

### Running the Project

1. Navigate to the project directory:


2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the app in your browser:

```bash
http://localhost:3000
```

### Building for Production

Run the production build:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```
